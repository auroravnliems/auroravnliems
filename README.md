## Featured Projects

### 🔧 AuroraCraft Plugin Suite

| Plugin | Description |
|---|---|
| **AuroraCombat** | Full PvP combat management — combat tag, PvP toggle, newbie protection, anti-border hopping, item cooldowns, kill rewards. Paper 1.21.1+ / Folia. |
| **AuroraDonate** | Donation card & bank system integrated with PlayerPoints, with differentiated reward rates. |
| **AuroraToken** | Custom token economy with auto-give scheduling and GUI shop. |
| **AuroraSecurity** | Plugin license enforcement with License4J, XOR-encoded credentials, and Polymart API validation. |
| **AuroraItemInfo** | In-game item inspection tool exposing MMOItems stats, Nexo metadata, and custom attributes. |
| **AuroraPunishNotify** | CMI punishment → Discord webhook bridge for real-time staff notifications. |
| **AuroraTooltip** | Wynncraft-style multi-page tooltip system with F-key page switching, tier glyphs, and stat icons. |

### 🌍 Server Infrastructure — AuroraCraft

- **Network:** `auroracraft.online` · Leaf 1.21.x · Pterodactyl on VPS (`118.69.187.178`)
- **Protection:** TCPShield + iptables FORWARD rules isolating Docker container ports
- **Panel:** Cloudflare-proxied Pterodactyl, SSH on port 2222
- **World Gen:** Terra packs (Origen, Tartarus, ReimagEND) with 1.21+ compatibility fixes
